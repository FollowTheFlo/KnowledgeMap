Main purpose of this App

  • Build Quizes from any Article of Wikipedia, so over 5 millions!
  • Test your knowledge on any Article of Wikipedia, so over 5 millions!
  • Quick process based on Multiple choices: it takes a minute to build and answer a quiz!
  • Optimized Shuffle: run quiz on same article multiple times, choices are always different!

'Enter Article' Menu

User writes a keyword that must match a Wikipedia Article Title.

It is cases/space sensitive, the spelling must be the EXACT same as the wikipedia Article Title, ex:

  • 'Smashing Pumpkins' -> Wrong
  • 'The smashing pumpkins' -> Wrong
  • 'The Smashing pumpkins' -> Wrong
  • 'The Smashing Pumpkins' -> Right

History Menu

List all the quizes since the application has been running, it is not saving data on the device hard drive.
If you reopen the app, the quiz history is cleared.

Tools Menu

  • Select the maximum number of choices for each question (minimum 2)
  • Select the maximum number of questions for the full quiz
  • Show Answer after each questions via a popup window
  • The changes are applied to the next quiz running

Questions based on Wikipedia Articles

The App generates Quiz Questions based on the entered wikipedia article. Using SPARQL over DBpedia, it queries the subjects associated to this article, it searchs for articles (Distractors) having its subjects in common, then, the distractors's subjects are returned.

Performances

The 'Max number of questions' set in Tools menu has an impact on the performance.
The delay is roughly proportional to this max questions number.
Decrease the value if the delay to generate the quiz is too long

Technology used

HTML,CSS,Javascript,Jquery,JQuery Mobile,Cordova,Notepad++,SPARQL,DBpedia
